# Install 
## Windows (powershell)
### SSH clone
```powershell
git clone --branch wezterm git@github.com:fexsecc/nvim-config.git $ENV:USERPROFILE\AppData\Local\nvim
```
### HTTPS clone
```powershell
git clone --branch wezterm https://github.com/fexsecc/nvim-config.git $ENV:USERPROFILE\AppData\Local\nvim
```
